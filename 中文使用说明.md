# OpenManus-GUI 中文使用说明

欢迎使用 OpenManus-GUI！这是一个带有现代化 Web 界面的 AI Agent 系统，支持对话历史、多轮对话和模型切换。

---

### 核心功能

-   **Gradio Web UI**: 在浏览器中与 Agent 交互，界面美观，操作直观。
-   **对话历史**: 所有对话都会自动保存，随时可以回顾。
-   **多会话管理**: 可以创建、重命名、删除不同的聊天会话。
-   **模型切换**: 支持一键在多种云端和本地模型之间切换。
-   **完整的 Agent 能力**: 保留了 OpenManus 核心的上网、写代码、操作文件等所有自动化能力。

---

### 安装与运行

详细的安装和运行步骤，请参考 [**RUNBOOK.md (运行手册)**](RUNBOOK.md)。

简单来说，您只需要：

1.  **获取文件**: 从本仓库 `git pull` 或下载 ZIP。
2.  **运行安装脚本**: 右键 `install_webui_windows.ps1` -> "使用 PowerShell 运行"。
3.  **启动 Web UI**: 双击 `E:\OpenManus\OpenManus-GUI\启动WebUI.bat`。
4.  **开始使用**: 在浏览器打开 `http://localhost:7860`。

---

### 目录结构说明

安装完成后，您的 `E:\OpenManus\OpenManus-GUI` 目录结构如下：

```
E:\OpenManus\OpenManus-GUI
├── .venv/                # Python 虚拟环境
├── app/                  # Agent 核心代码
├── config/
│   └── config.toml       # 配置文件 (API Key, 模型)
├── chatsHistory/         # 对话历史记录
├── workspace/            # Agent 的工作目录 (生成的文件都在这里)
├── 启动WebUI.bat         # 启动 Gradio 界面 (推荐)
├── 启动终端模式.bat      # 启动经典终端界面
├── 启动API服务.bat       # 启动 OpenAI 兼容的 API 服务
└── 切换模型.bat          # 切换 LLM 模型
```
