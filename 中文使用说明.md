## 1. 首次安装

如果您是第一次使用，请根据您的操作系统运行对应的**一键安装脚本**。这些脚本会自动完成所有环境配置。

| 操作系统 | 运行脚本 |
|---|---|
| **Windows** | `install_windows.ps1` |
| **macOS / Linux** | `install_unix.sh` |

**安装步骤**：

1.  **下载仓库文件**：将本项目所有文件下载到您的电脑。
2.  **运行脚本**：
    -   **Windows**: 右键点击 `install_windows.ps1` → 选择 “**使用 PowerShell 运行**”。
    -   **macOS/Linux**: 打开终端，进入项目目录，执行 `chmod +x install_unix.sh && ./install_unix.sh`。
3.  **输入 API Key**：脚本会提示您输入 OpenAI API Key。请前往 [platform.openai.com/api-keys](https://platform.openai.com/api-keys) 创建一个。**强烈建议不要跳过此步**。

安装完成后，所有必需的文件（包括 OpenManus 源码）都会被放置在您用户目录下的 `OpenManus` 文件夹中。

## 2. 日常使用

安装完成后，您可以通过以下方式启动 AI Agent：

| 操作系统 | 启动方式 |
|---|---|
| **Windows** | 双击 `启动Agent.bat` 文件 |
| **macOS / Linux** | 在终端执行 `start.sh` 文件 |

> **提示**：您可以将这些启动脚本的快捷方式放到桌面，方便快速访问。

启动后，一个终端窗口会打开，您就可以在里面输入您的任务目标，开始与 AI Agent 交互了。

## 3. 切换模型

本方案最大的特色就是可以灵活切换驱动 Agent 的“大脑”（LLM 模型），以平衡性能和成本。

**如何切换**：

| 操作系统 | 操作方式 |
|---|---|
| **Windows** | 双击 `切换模型.bat` 文件 |
| **macOS / Linux** | 在终端执行 `python switch_model.py` (需先激活虚拟环境) |

运行后，您会看到一个清晰的菜单，列出了所有预设模型及其特点。输入数字即可一键切换。

**预设模型对比**：

| 模型 | 单次任务费用 | 推荐场景 |
|---|---|---|
| **GPT-4.1-mini** | **¥0.14** | 日常任务（**推荐默认**） |
| **GPT-5-mini** | ¥0.25 | 需要更强推理时 |
| **GPT-5.2** | ¥0.86 | 复杂编码/重要任务 |
| **Ollama 本地模型** | **¥0 (免费)** | 隐私敏感/离线使用 |

### 使用本地 Ollama 模型

如果您的电脑有不错的独立显卡（如 RTX 5070），可以尝试完全免费、100% 本地运行的方案。

1.  **安装 Ollama**: 从 [ollama.com/download](https://ollama.com/download) 下载并安装 Ollama。
2.  **下载模型**: 在终端执行 `ollama pull qwen2.5:14b` (推荐) 或 `ollama pull llama3.3:8b`。
3.  **切换模型**: 运行模型切换工具，选择对应的 Ollama 模型即可。

## 4. 常见问题 (FAQ)

**Q: OpenAI API Key 在哪里获取？收费吗？**

A: 您可以用您的 ChatGPT Plus 账号登录 [platform.openai.com/api-keys](https://platform.openai.com/api-keys) 创建。API 是独立于 Plus 订阅计费的，但新注册用户有 **$5 免费额度**，使用 GPT-4.1-mini 的话，足够免费用很久了。

**Q: 为什么 `config.toml` 文件不见了？**

A: 为了保护您的 API Key 安全，我们已经将 `config/config.toml` 文件添加到了 `.gitignore` 中。这意味着它不会被上传到 GitHub。安装脚本会自动在您的本地 `OpenManus` 目录中创建这个文件。

**Q: 启动时报错怎么办？**

A: 请确保您已经按照安装脚本的指引正确安装了所有依赖。如果问题仍然存在，可以尝试删除 `OpenManus` 文件夹，然后重新运行一次安装脚本。

**Q: 我可以自己添加其他模型吗？**

A: 当然可以！运行模型切换工具时，选择“自定义模型”，然后输入模型的名称、API Base URL 和 API Key 即可。
